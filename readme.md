Installation:

install anaconda
run anaconda prompt

run:
conda create --name "Timbrer"
conda install -c conda-forge librosa
conda install -c conda-forge python-sounddevice

modify data\soundfont\8MBGMSFX.cfg to contain the correct absolute path
